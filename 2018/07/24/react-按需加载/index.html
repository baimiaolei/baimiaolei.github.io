react--按需加载

发表于 2018-07-24

我又来啦。
页面加载慢，需要做按需加载，万万没想到，react-router4的按需加载，不能用webpack以往

const index = r => require.ensure([], () => r(require('../components/index')), 'index')
然后，查资料，提供了一个思路，使用bundle,但是出现了一个问题，this.props.history,找不到。。。
后来找到一个方法，使用withRouter,可是我看不懂，也不会用。

算了，不废话了，祭出终级神器，prop-types
这个可以用npm install prop-types 安装.
然后在需要使用this.props.history的界面，在constructor上面这样写：
static contextTypes = { router: PropTypes.object.isRequired };
然后使用 this.context.router.history 就可以跳转了 ^_^ ~
